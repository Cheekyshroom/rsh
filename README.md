# rune-shell
A more traditional way of navigating unix, it should support any VT100 escape sequences.

# example usage
```
> rune-shell
ᛈᛖᛏᛖᚱᛋ-ᛗᛒᛈ:ᚱᛋᚻ ᛈᛖᛏᛖᚱᚻᛖᚱᚾᛁᛗᚨᚾ$ ᚠᚩᚱ ᛁ ᛁᚾ `ᛋᛖᛣ 1 10`; ᛞᚩ ᛖᚳᚩ "ᛁᚾᛞᛖᛉ ᛁᛋ: $ᛁ"; ᛞᚩᚾᛖ
ᛁᚾᛞᛖᛉ ᛁᛋ: 1
ᛁᚾᛞᛖᛉ ᛁᛋ: 2
ᛁᚾᛞᛖᛉ ᛁᛋ: 3
ᛁᚾᛞᛖᛉ ᛁᛋ: 4
ᛁᚾᛞᛖᛉ ᛁᛋ: 5
ᛁᚾᛞᛖᛉ ᛁᛋ: 6
ᛁᚾᛞᛖᛉ ᛁᛋ: 7
ᛁᚾᛞᛖᛉ ᛁᛋ: 8
ᛁᚾᛞᛖᛉ ᛁᛋ: 9
ᛁᚾᛞᛖᛉ ᛁᛋ: 10
ᛈᛖᛏᛖᚱᛋ-ᛗᛒᛈ:ᚱᛋᚻ ᛈᛖᛏᛖᚱᚻᛖᚱᚾᛁᛗᚨᚾ$
```
